# Projeto de Leitor de arquivos CSV

O arquivo csv aceito é proveniente do site DATASUS: https://datasus.saude.gov.br/acesso-a-informacao/doencas-e-agravos-de-notificacao-de-2007-em-diante-sinan/
Nesse endereço, será necessário baixar o arquivo csv das notificações de doença na abrangência geográfica de "Brasil por Região, UF e Municipio"

Antes de rodar usando o Package docker, coloque a variável de ambiente DEBUG com valor True

Dentro do sistema, faça o upload do arquivo e seja feliz!
